## Travel Club Project

여행 클럽과 클럽 멤버의 관리(클럽 개설과 변경 등)를 위한 app

- 크게 4단계의 과정을 통해 개발
- 웹 환경에서 서비스 할 수 있도록 구현
- UI는 REST-Client(insomnia)를 이용
